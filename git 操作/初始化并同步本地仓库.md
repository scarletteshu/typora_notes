- 初始化仓库

```bash
cd filedir
git init
```

- 将项目所有文件添加到仓库

```bash
git add .
```

- 添加README.md（先新建）

```bash
git add README.md
```

- 提交到本地仓库

```bash
git commit -m "comment words"
```

- 本地仓库关联github（仓库要同名）

```bash
git remote add origin https://github.com/xxx/xxx
```

- pull request

```bash
git pull origin master
```

- upload

```bash
git push -u origin master
```

> 可能会要求天username 和password
